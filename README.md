# Login Age Guesser (Pygame)

An interactive UI-based Python application where the system guesses the user's date of birth using a narrowing range algorithm.

## How It Works
- The program generates random dates
- The user responds with:
  - "Earlier"
  - "Later"
  - "Correct"
- The system narrows down the possible range until it finds the correct date

## Features
- Interactive UI with input fields
- Mouse-based controls
- Dynamic date range narrowing (binary search logic)
- Visual feedback system

## Built With
- Python
- Pygame
